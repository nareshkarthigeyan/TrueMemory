## لماذا TrueMemory

**يجد الإشارة وسط الضوضاء.** ذكاؤك الاصطناعي يرى آلاف الرسائل. TrueMemory يحدد أيها مهم فعلاً، ويتجاهل الباقي. بدون وسوم يدوية، بدون هندسة موجهات. إنه ببساطة يعرف.

**يصبح أكثر دقة مع الوقت.** إنه ليس قاعدة بيانات ثابتة. إنها ذاكرة حية تنمو معك، تحل التناقضات عندما تغير رأيك، وتحدث الحقائق القديمة، وتدمج ما تعرفه. كلما استخدمته أكثر، أصبح أفضل.

**يعمل دون أن تفكر فيه.** يلتقط TrueMemory الذكريات تلقائياً من محادثاتك ويحقن الذكريات الصحيحة تلقائياً في جلستك التالية. لن تحتاج أبداً لتخزين أو البحث عن أي شيء يدوياً. كل شيء يحدث تلقائياً.

**محلي 100%.** ملف SQLite واحد على جهازك. لا شيء يغادر جهازك. بدون سحابة، بدون حاجة لمفاتيح API. بياناتك ملكك.

> **بدون TrueMemory:** "ما هو إطار العمل الذي نستخدمه؟" سُئل للمرة الثانية عشرة هذا الأسبوع. وكيلك يبدأ كل جلسة بفقدان ذاكرة. لا يعرف اسمك، ولا مجموعة أدواتك، ولا أي شيء أخبرته به بالأمس.
>
> **مع TrueMemory:** وكيلك يعرف بالفعل أنك تستخدم FastAPI، وتفضل Pydantic v2، وأن وسيط المصادقة الخاص بك في `src/auth/`. يتذكر تصحيحاتك وتفضيلاتك وقراراتك. عبر كل جلسة، إلى الأبد.

## البدء السريع

### لـ Claude Code / Claude CLI / Cursor / Codex CLI / Gemini CLI

```bash
curl -LsSf https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.sh | sh
```

> يثبت كل شيء في بيئة معزولة. يحمل حوالي 1.5 غيغابايت من نماذج الذكاء الاصطناعي. لا تغادر أي بيانات جهازك. لا يحتاج sudo.

<details><summary>جديد على الطرفية؟ انقر هنا للحصول على تعليمات خطوة بخطوة.</summary>

1. **افتح طرفية:** Mac: `Cmd + Space`، اكتب `Terminal`. Linux: `Ctrl + Alt + T`. Windows: افتح PowerShell.
2. **الصق الأمر أعلاه** واضغط Enter.
3. **انتظر 3-5 دقائق** لتحميل النماذج.
4. **أغلق أداة الذكاء الاصطناعي بالكامل** وأعد فتحها (Mac: `Cmd+Q`).
5. **اكتب "Set up TrueMemory"** واختر مستوى.

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.ps1 | iex
```

</details>

هذا كل شيء. من الآن فصاعداً، يتذكر TrueMemory محادثاتك تلقائياً.

## المستويات

نفس البنية، ثلاثة مستويات. جميعها مضمنة في تثبيت واحد. بدّل في أي وقت بقول "switch to Pro" أو "switch to Base".

| | Edge | Base | Pro |
|---|------|------|-----|
| **LoCoMo** | 89.6% | 92.0% | **93.0%** |
| **LongMemEval** | — | — | **92.0%** |
| **BEAM-1M** | — | — | **76.6% (SOTA)** |
| **نموذج التضمين** | 8 ميغابايت خفيف | 600 ميغابايت عالي الدقة | 600 ميغابايت عالي الدقة |
| **معيد الترتيب** | 22M معامل | 149M معامل | 149M معامل |
| **بحث HyDE** | — | — | ✅ (يتطلب مفتاح API لـ LLM) |
| **يعمل على** | أي جهاز، CPU فقط | 4 غيغابايت+ ذاكرة | 4 غيغابايت+ ذاكرة + مفتاح API |

**Edge** يعمل في كل مكان. **Base** هو أقوى مستوى يعمل بالكامل دون اتصال. **Pro** يضيف توسيع استعلامات بالذكاء الاصطناعي للحصول على أعلى الدرجات.

## Python API

```python
from truememory import Memory

m = Memory()

m.add("Prefers dark mode and TypeScript", user_id="alex")
m.add("Works at Anthropic as a senior engineer", user_id="alex")

results = m.search("What are Alex's preferences?", user_id="alex")
results = m.search_deep("career history?", user_id="alex")  # multi-round, higher accuracy
```

| الطريقة | الوصف |
|--------|-------------|
| `m.add(content, user_id)` | تخزين ذاكرة |
| `m.search(query, user_id)` | البحث (خط أنابيب من 6 طبقات + معيد ترتيب) |
| `m.search_deep(query, user_id)` | بحث وكيلي متعدد الجولات |
| `m.get(id)` / `m.get_all(user_id)` | استرجاع الذكريات |
| `m.update(id, content)` / `m.delete(id)` | تعديل أو حذف |
| `m.stats()` | إحصائيات النظام |

[مرجع API الكامل →](../docs/python-api.md)

## الأسئلة الشائعة

<details><summary><strong>أين تُخزن بياناتي؟ هل يُرسل أي شيء إلى السحابة؟</strong></summary>

كل شيء يُخزن محلياً في `~/.truememory/memories.db`. مستويا Edge و Base لا يجريان أي اتصالات خارجية. Pro يرسل فقط نص استعلام البحث الخاص بك إلى LLM لتوسيع الاستعلام. ذكرياتك لا تُنقل أبداً.
</details>

<details><summary><strong>هل أحتاج إلى تثبيت Python؟</strong></summary>

لا. المثبت يستخدم [uv](https://docs.astral.sh/uv/) لإدارة Python 3.12 معزول. لا يُمس Python الخاص بنظامك أبداً.
</details>

<details><summary><strong>لماذا لا نستخدم نافذة سياق أكبر فحسب؟</strong></summary>

نوافذ السياق مكلفة وبطيئة وفارغة في بداية كل جلسة. TrueMemory يوفر استدعاءً فورياً بصفر رموز سياق، في أقل من 200 مللي ثانية.
</details>

<details><summary><strong>هل يجمع TrueMemory بيانات القياس عن بُعد؟</strong></summary>

القياس عن بُعد للاستخدام المجهول (استدعاءات الأدوات، عدد الجلسات، معلومات المنصة) مُفعّل افتراضياً. نحن **لا** نتتبع أبداً محتوى الذاكرة أو الاستعلامات أو مسارات الملفات أو مفاتيح API. لإلغاء الاشتراك: `export TRUEMEMORY_TELEMETRY=off`
</details>
