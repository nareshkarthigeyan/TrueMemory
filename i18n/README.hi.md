## TrueMemory क्यों

**यह शोर में से संकेत खोजता है।** आपका AI हज़ारों संदेश देखता है। TrueMemory पता लगाता है कि कौन से वास्तव में महत्वपूर्ण हैं, और बाकी को हटा देता है। कोई मैनुअल टैगिंग नहीं, कोई प्रॉम्प्ट इंजीनियरिंग नहीं। यह बस जानता है।

**यह समय के साथ और तेज़ होता जाता है।** यह कोई स्थिर डेटाबेस नहीं है। यह एक जीवित मेमोरी है जो आपके साथ बढ़ती है, जब आप अपना मन बदलते हैं तो विरोधाभासों को सुलझाती है, पुराने तथ्यों को अपडेट करती है, और जो जानती है उसे समेकित करती है। जितना अधिक आप इसका उपयोग करेंगे, उतना बेहतर यह होगा।

**यह बिना आपकी सोच के काम करता है।** TrueMemory स्वचालित रूप से आपकी बातचीत से यादें कैप्चर करता है और अगले सत्र में स्वचालित रूप से सही यादें इंजेक्ट करता है। आपको कभी मैन्युअल रूप से कुछ भी स्टोर या खोजना नहीं पड़ता। यह बस होता है।

**यह 100% लोकल है।** आपकी मशीन पर एक SQLite फ़ाइल। कोई भी डेटा आपके डिवाइस से बाहर नहीं जाता। कोई क्लाउड नहीं, कोई API कुंजी की ज़रूरत नहीं। आपका डेटा आपका है।

> **TrueMemory के बिना:** "हम कौन सा फ्रेमवर्क इस्तेमाल कर रहे हैं?" इस हफ़्ते 12वीं बार पूछा गया। आपका एजेंट हर सत्र भूलने की बीमारी के साथ शुरू करता है। उसे आपका नाम, आपका स्टैक, या कल आपने जो बताया उसमें से कुछ भी याद नहीं है।
>
> **TrueMemory के साथ:** आपका एजेंट पहले से जानता है कि आप FastAPI इस्तेमाल करते हैं, Pydantic v2 पसंद करते हैं, और आपका auth middleware `src/auth/` में है। यह आपके सुधारों, आपकी प्राथमिकताओं और आपके निर्णयों को याद रखता है। हर सत्र में, हमेशा के लिए।

## त्वरित शुरुआत

### Claude Code / Claude CLI / Cursor / Codex CLI / Gemini CLI के लिए

```bash
curl -LsSf https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.sh | sh
```

> सब कुछ एक पृथक वातावरण में इंस्टॉल करता है। ~1.5GB AI मॉडल डाउनलोड करता है। कोई डेटा आपकी मशीन से बाहर नहीं जाता। sudo की ज़रूरत नहीं।

<details><summary>टर्मिनल में नए हैं? चरण-दर-चरण निर्देशों के लिए यहाँ क्लिक करें।</summary>

1. **टर्मिनल खोलें:** Mac: `Cmd + Space`, `Terminal` टाइप करें। Linux: `Ctrl + Alt + T`। Windows: PowerShell खोलें।
2. **ऊपर दिया गया कमांड पेस्ट करें** और Enter दबाएँ।
3. **3-5 मिनट प्रतीक्षा करें** मॉडल डाउनलोड होने के लिए।
4. **अपना AI टूल पूरी तरह बंद करें** और फिर से खोलें (Mac: `Cmd+Q`)।
5. **"Set up TrueMemory" टाइप करें** और एक टियर चुनें।

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.ps1 | iex
```

</details>

| **LoCoMo** | 89.6% | 92.0% | **93.0%** |

## Python API

```python
from truememory import Memory

m = Memory()

m.add("Prefers dark mode and TypeScript", user_id="alex")
m.add("Works at Anthropic as a senior engineer", user_id="alex")

results = m.search("What are Alex's preferences?", user_id="alex")
results = m.search_deep("career history?", user_id="alex")  # multi-round, higher accuracy
```

| विधि | विवरण |
|--------|-------------|
| `m.add(content, user_id)` | एक मेमोरी स्टोर करें |
| `m.search(query, user_id)` | खोजें (6-लेयर पाइपलाइन + रीरैंकर) |
| `m.search_deep(query, user_id)` | मल्टी-राउंड एजेंटिक सर्च |
| `m.get(id)` / `m.get_all(user_id)` | मेमोरी प्राप्त करें |
| `m.update(id, content)` / `m.delete(id)` | संशोधित या हटाएँ |
| `m.stats()` | सिस्टम आँकड़े |

[पूर्ण API संदर्भ →](../docs/python-api.md)

## अक्सर पूछे जाने वाले प्रश्न

<details><summary><strong>मेरा डेटा कहाँ संग्रहित होता है? क्या कुछ क्लाउड पर भेजा जाता है?</strong></summary>

सब कुछ स्थानीय रूप से `~/.truememory/memories.db` में रहता है। Edge और Base टियर कोई बाहरी कॉल नहीं करते। Pro केवल क्वेरी विस्तार के लिए आपके सर्च क्वेरी टेक्स्ट को LLM को भेजता है। आपकी यादें कभी प्रसारित नहीं होतीं।
</details>

<details><summary><strong>क्या मुझे Python इंस्टॉल करना होगा?</strong></summary>

नहीं। इंस्टॉलर सैंडबॉक्स्ड Python 3.12 को प्रबंधित करने के लिए [uv](https://docs.astral.sh/uv/) का उपयोग करता है। आपका सिस्टम Python कभी नहीं छुआ जाता।
</details>

<details><summary><strong>बड़ी कॉन्टेक्स्ट विंडो का उपयोग क्यों नहीं करते?</strong></summary>

कॉन्टेक्स्ट विंडो महँगी, धीमी होती हैं, और हर सत्र की शुरुआत में खाली होती हैं। TrueMemory शून्य कॉन्टेक्स्ट टोकन में, 200ms से कम में तत्काल रिकॉल देता है।
</details>

<details><summary><strong>क्या TrueMemory टेलीमेट्री एकत्र करता है?</strong></summary>

अनाम उपयोग टेलीमेट्री (टूल कॉल, सत्र गणना, प्लेटफ़ॉर्म जानकारी) डिफ़ॉल्ट रूप से चालू है। हम **कभी भी** मेमोरी सामग्री, क्वेरी, फ़ाइल पथ, या API कुंजियाँ ट्रैक नहीं करते। ऑप्ट आउट करें: `export TRUEMEMORY_TELEMETRY=off`
</details>
