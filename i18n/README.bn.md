## কেন TrueMemory

**এটি শব্দের মধ্যে থেকে সংকেত খুঁজে বের করে।** আপনার AI হাজার হাজার বার্তা দেখে। TrueMemory বুঝে নেয় কোনগুলো আসলে গুরুত্বপূর্ণ এবং বাকিগুলো বাদ দিয়ে দেয়। কোনো ম্যানুয়াল ট্যাগিং নেই, কোনো প্রম্পট ইঞ্জিনিয়ারিং নেই। এটি নিজেই বোঝে।

**এটি সময়ের সাথে আরো শক্তিশালী হয়।** এটি কোনো স্থির ডাটাবেস নয়। এটি একটি জীবন্ত মেমরি যা আপনার সাথে বেড়ে ওঠে। আপনি মত পরিবর্তন করলে দ্বন্দ্ব সমাধান করে, পুরনো তথ্য আপডেট করে, এবং যা জানে তা একত্রিত করে। আপনি যত বেশি ব্যবহার করবেন, এটি তত ভালো হবে।

**এটি আপনার চিন্তা ছাড়াই কাজ করে।** TrueMemory স্বয়ংক্রিয়ভাবে আপনার কথোপকথন থেকে স্মৃতি সংগ্রহ করে এবং স্বয়ংক্রিয়ভাবে আপনার পরবর্তী সেশনে সঠিক স্মৃতিগুলো ইনজেক্ট করে। আপনাকে কখনোই ম্যানুয়ালি কিছু সংরক্ষণ বা অনুসন্ধান করতে হবে না। এটি এমনিতেই ঘটে।

**এটি ১০০% লোকাল।** আপনার মেশিনে একটি মাত্র SQLite ফাইল। কিছুই আপনার ডিভাইস ছেড়ে যায় না। কোনো ক্লাউড নেই, কোনো API কী প্রয়োজন নেই। আপনার ডেটা আপনার।

> **TrueMemory ছাড়া:** "আমরা কোন ফ্রেমওয়ার্ক ব্যবহার করছি?" এই সপ্তাহে ১২ বার জিজ্ঞাসা করা হয়েছে। আপনার এজেন্ট প্রতিটি সেশন স্মৃতিভ্রংশ নিয়ে শুরু করে। এটি আপনার নাম, আপনার স্ট্যাক, বা গতকাল আপনি যা বলেছিলেন কিছুই জানে না।
>
> **TrueMemory দিয়ে:** আপনার এজেন্ট ইতিমধ্যে জানে আপনি FastAPI ব্যবহার করেন, Pydantic v2 পছন্দ করেন, এবং আপনার auth middleware `src/auth/`-এ আছে। এটি আপনার সংশোধন, আপনার পছন্দ, এবং আপনার সিদ্ধান্তগুলো মনে রাখে। প্রতিটি সেশন জুড়ে, চিরকাল।

## দ্রুত শুরু

### Claude Code / Claude CLI / Cursor / Codex CLI / Gemini CLI-এর জন্য

```bash
curl -LsSf https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.sh | sh
```

> একটি বিচ্ছিন্ন পরিবেশে সবকিছু ইনস্টল করে। ~1.5GB AI মডেল ডাউনলোড করে। কোনো ডেটা আপনার মেশিন ছেড়ে যায় না। sudo প্রয়োজন নেই।

<details><summary>টার্মিনালে নতুন? ধাপে ধাপে নির্দেশনার জন্য এখানে ক্লিক করুন।</summary>

1. **একটি টার্মিনাল খুলুন:** Mac: `Cmd + Space`, `Terminal` টাইপ করুন। Linux: `Ctrl + Alt + T`। Windows: PowerShell খুলুন।
2. **উপরের কমান্ডটি পেস্ট করুন** এবং Enter চাপুন।
3. **মডেল ডাউনলোডের জন্য ৩-৫ মিনিট অপেক্ষা করুন।**
4. **আপনার AI টুলটি সম্পূর্ণ বন্ধ করুন** এবং পুনরায় খুলুন (Mac: `Cmd+Q`)।
5. **"Set up TrueMemory" টাইপ করুন** এবং একটি টিয়ার বেছে নিন।

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/buildingjoshbetter/TrueMemory/main/install.ps1 | iex
```

## টিয়ার

একই আর্কিটেকচার, তিনটি টিয়ার। একটি মাত্র ইনস্টলেশনে সবই অন্তর্ভুক্ত। যেকোনো সময় "switch to Pro" বা "switch to Base" বলে পরিবর্তন করুন।

| | Edge | Base | Pro |
|---|------|------|-----|
| **LoCoMo** | 89.6% | 92.0% | **93.0%** |
| **LongMemEval** | — | — | **92.0%** |
| **BEAM-1M** | — | — | **76.6% (SOTA)** |
| **এম্বেডিং মডেল** | 8 MB হালকা | 600 MB উচ্চ-নির্ভুলতা | 600 MB উচ্চ-নির্ভুলতা |
| **রির্যাংকার** | 22M প্যারামিটার | 149M প্যারামিটার | 149M প্যারামিটার |
| **HyDE সার্চ** | — | — | ✅ (LLM API কী প্রয়োজন) |
| **চলে** | যেকোনো মেশিনে, শুধু CPU | 4 GB+ RAM | 4 GB+ RAM + API কী |

**Edge** সব জায়গায় কাজ করে। **Base** হলো সম্পূর্ণ অফলাইনে সবচেয়ে শক্তিশালী টিয়ার। **Pro** সর্বোচ্চ স্কোরের জন্য AI-চালিত কোয়েরি সম্প্রসারণ যোগ করে।

## Python API

```python
from truememory import Memory

m = Memory()

m.add("Prefers dark mode and TypeScript", user_id="alex")
m.add("Works at Anthropic as a senior engineer", user_id="alex")

results = m.search("What are Alex's preferences?", user_id="alex")
results = m.search_deep("career history?", user_id="alex")  # multi-round, higher accuracy
```

| মেথড | বিবরণ |
|--------|-------------|
| `m.add(content, user_id)` | একটি মেমরি সংরক্ষণ করুন |
| `m.search(query, user_id)` | সার্চ (৬-স্তর পাইপলাইন + রির্যাংকার) |
| `m.search_deep(query, user_id)` | মাল্টি-রাউন্ড এজেন্টিক সার্চ |
| `m.get(id)` / `m.get_all(user_id)` | মেমরি পুনরুদ্ধার করুন |
| `m.update(id, content)` / `m.delete(id)` | পরিবর্তন বা মুছুন |
| `m.stats()` | সিস্টেম পরিসংখ্যান |

[সম্পূর্ণ API রেফারেন্স →](../docs/python-api.md)

## প্রায়শই জিজ্ঞাসিত প্রশ্ন

<details><summary><strong>আমার ডেটা কোথায় সংরক্ষিত হয়? কিছু কি ক্লাউডে পাঠানো হয়?</strong></summary>

সবকিছু লোকালি `~/.truememory/memories.db`-তে থাকে। Edge এবং Base টিয়ার কোনো বাহ্যিক কল করে না। Pro শুধুমাত্র কোয়েরি সম্প্রসারণের জন্য আপনার সার্চ কোয়েরি টেক্সট একটি LLM-এ পাঠায়। আপনার মেমরি কখনোই প্রেরিত হয় না।
</details>

<details><summary><strong>আমার কি Python ইনস্টল করা দরকার?</strong></summary>

না। ইনস্টলার [uv](https://docs.astral.sh/uv/) ব্যবহার করে একটি স্যান্ডবক্সড Python 3.12 পরিচালনা করে। আপনার সিস্টেম Python কখনো স্পর্শ করা হয় না।
</details>

<details><summary><strong>বড় কনটেক্সট উইন্ডো ব্যবহার করলেই হয় না?</strong></summary>

কনটেক্সট উইন্ডো ব্যয়বহুল, ধীর, এবং প্রতিটি সেশনের শুরুতে খালি থাকে। TrueMemory শূন্য টোকেন কনটেক্সটে, ২০০ms-এর কম সময়ে তাৎক্ষণিক রিকল প্রদান করে।
</details>

<details><summary><strong>TrueMemory কি টেলিমেট্রি সংগ্রহ করে?</strong></summary>

বেনামী ব্যবহারের টেলিমেট্রি (টুল কল, সেশন গণনা, প্ল্যাটফর্ম তথ্য) ডিফল্টভাবে চালু থাকে। আমরা **কখনোই** মেমরি বিষয়বস্তু, কোয়েরি, ফাইল পাথ, বা API কী ট্র্যাক করি না। বন্ধ করতে: `export TRUEMEMORY_TELEMETRY=off`
</details>
